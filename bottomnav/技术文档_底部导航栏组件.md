# 底部导航栏组件技术文档

## 1. 组件介绍

`BottomNavBar` 用于在页面底部展示多个导航入口，支持：
- 图标 + 文字
- 选中状态高亮
- 选中指示条
- 点击回调

## 2. 模块结构

```
bottomnav/
├─ build.gradle
├─ proguard-rules.pro
└─ src/main/
   ├─ AndroidManifest.xml
   └─ java/com/example/bottomnav/
      ├─ BottomNavBar.java
      └─ BottomNavItem.java
```

## 3. 核心类说明

- `BottomNavBar`
  - 主视图容器
  - 提供 `setItems()`、`setSelectedIndex()` 等接口
- `BottomNavItem`
  - 单个导航项数据模型（标题 + 可选图标）

## 4. 使用方式

### 4.1 依赖引入

在 `settings.gradle` 中添加：
```
include ':bottomnav'
```

在 `app/build.gradle` 中添加：
```
implementation project(':bottomnav')
```

### 4.2 XML 使用

```xml
<com.example.bottomnav.BottomNavBar
    android:id="@+id/bottom_nav"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    app:indicatorVisible="false" />
```

### 4.3 Java 使用

```java
BottomNavBar nav = findViewById(R.id.bottom_nav);
List<BottomNavItem> items = new ArrayList<>();
items.add(new BottomNavItem("首页"));
items.add(new BottomNavItem("发现"));
items.add(new BottomNavItem("消息"));
items.add(new BottomNavItem("我的"));
nav.setItems(items);

nav.setOnItemSelectedListener(new BottomNavBar.OnItemSelectedListener() {
    @Override
    public void onItemSelected(int index, BottomNavItem item) {
        // 切换页面
    }

    @Override
    public void onItemReselected(int index, BottomNavItem item) {
        // 重复点击
    }
});
```

## 5. 可配置项

```java
nav.setSelectedColor(0xFF1F1F1F);
nav.setUnselectedColor(0xFF888888);
nav.setIndicatorColor(0xFF1F1F1F);
nav.setIndicatorHeightDp(3);
nav.setTextSizeSp(12f);
nav.setIndicatorVisible(false);
```

### 5.1 可用 XML 属性

- `app:indicatorVisible` 是否显示指示条
- `app:indicatorHeight` 指示条高度
- `app:selectedColor` 选中文字/图标颜色
- `app:unselectedColor` 未选中文字/图标颜色
- `app:indicatorColor` 指示条颜色
- `app:textSize` 文字大小

## 6. AAR 打包

```
./gradlew :bottomnav:assembleRelease
```

输出路径：
```
bottomnav/build/outputs/aar/bottomnav-release.aar
```
